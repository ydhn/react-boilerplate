# React 16.3 Boilerplate

### Install
Install dependencies.
```sh
$ npm install
```

### Start
This script runs the webpack-dev-server that provides live reloading.
```sh
$ npm start
```
The default port of this server is 5000.

### Test
test with jest
```sh
$ npm run test
```

### Build
build production
```sh
$ npm run dist
```
